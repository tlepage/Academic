Note on installation of RAPS.
Please follow the instructions below:

1. Copy all contents of the CD to a directory of your choosing.
2. You must set the ODBC source of your operating system to final.mdb and to do this you must first go to Start->Settings->Control Panel
3. Then click on Administrative Tools
4. Click on Data Sources(ODBC)
5. Click "Add..."
6. Highlight "Microsoft Access Driver(*.mdb)"
7. Click "Finish"
8. In the Data Source Name dialog enter "final"
9. In the Database area of this page, click on Select and find the directory in which you 	installed the program and database.
10. Select "final.mdb" in the chooser dialog and click "OK"
11. Exit out and then double-click on RAPS.jar to start the program.
